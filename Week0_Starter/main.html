Hi! Welcome to FizzBuzz

This web application is a fun twist on the FizzBuzz word game.

The rules are as follows:

  • For a word length divisible by 3, we will display "Fizz".
  • For a word length divisible by 5, we will display "Buzz".
  • For a word length divisible by 3 and 5, we will display "FizzBuzz".
